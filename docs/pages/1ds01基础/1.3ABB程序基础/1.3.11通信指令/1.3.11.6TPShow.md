# 1.3.11.6 TPShow

## 1.含义

机器人在示教器屏幕上显示相应界面，通常情况下与机器人附加软件 Screen Viewer 配合使用。

## 2.用法

```
TPShow Window;

Window:	显示相应示教器窗口。	( tpnum )
```

- TP_PROGRAM 

  自动模式下，显示生产窗口。

  手动模式下，显示测试窗口。

- TP_LATEST 

  显示当前窗口的前一个窗口。

- TP_SCREENVIEWER 

  显示 Screen Viewer 窗口，需要相应软件。

实例：

```
…
TPShow TP_PROGRAM;
TPShow TP_LATEST;
…
```

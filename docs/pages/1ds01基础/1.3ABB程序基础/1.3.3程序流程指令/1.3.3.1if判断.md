# 1.3.3.1 if判断

1.含义

if意味“如果”，当前指令通过判断相应条件，控制需要执行的相应指令，是机器人程序流程基本指令。

如果满足判断条件，则执行if指令内部的程序，如果不满足，则跳过if指令内部的程序。

2.用法

程序结构为：

```rapid
if 判断条件 then
	指令1
elseif 判断条件 then
	指令2
elseif 判断条件 then
	指令3
else
	指令4
endif
```

if指令的判断条件后以then连接后续指令，最后以endif结尾。如果有多个分支判断，则可以添加elseif与判断条件。如果所有的分支都不满足，则会运行else后面的指令。

在if指令的结构中，elseif和else都是可选项，如果没有else，则一旦不满足上面的调节，就会跳过if段程序，执行后续的指令。

实例：

```rapid
if reg1>5 then
	Set do1;
	Set do2;
endif

if reg1>5 then
	Set do1;
	Set do2;
else
	Reset do1;
	Reset do2;
endif
	
if reg2=1 then
	routine1;
elseif reg2=2 then
	routine2;
elseif reg2=3 then
	routine3;
elseif reg2=4 then
	routine4;
else
	Error;
endif
```
